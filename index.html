Week 8 Homework: Open Refine

Using the Full Knight Grants Data Set and Open Refine, I was able to clean and analyze this data set to reveal which communities were benefiting the most from the Knight Scholarship Fund.

I first had to clean the data, which required going through the community and grantee location columns to see if the locations were spelt the correct way, and that there were no repeats. This was done using the 'cluster' function on Open Refine, after I had filtered each column by a text facet.

The next part that required cleaning up was the dates for when the grant was awarded, when the grant period began, and when the grant period ended. To clean these three columns up, I first chose to transform them to 'text cells', and then transformed them to 'date cells'. By doing this, the dates would all be consistent and correctly formatted.

I then needed filter the data by the communities that were to be benefited by the Knight Scholarship. To do this, I chose the 'community column' and then faceted it by text. I removed all the blank entries from the data, as those scholarships were ones that were not specifically benefiting a community.

I then needed to get the correct coordinates for the communities for when I exported the sheet for map building on CartoDB. To get the longitude and latitude, I had to connect the location column to a Google maps URL and then use a query to separate the location into latitude and longitude.
